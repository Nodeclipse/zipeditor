About the Zip-Editor

February 3rd, 2006

Abstract

The Zip-Editor is an Eclipse editor that can view and manipulate zip archives.

Features

It supports:

  • Drag-n-Drop from within the workbench or from outside applications
  • Viewing and manipulation of files from within an archive
  • Tree- and table view mode
